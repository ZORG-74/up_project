Менеджер личных заметок
1. Цель проекта
Веб-приложение для создания и управления личными заметками с авторизацией пользователей.

2. Функциональные требования
Пользователи
Регистрация (email, пароль, имя)

Вход и выход из системы

Только авторизованные видят свои заметки

Заметки
Создание (заголовок + текст)

Редактирование

Удаление (с подтверждением)

Просмотр списка всех своих заметок

Просмотр отдельной заметки полностью

Поиск и сортировка
Поиск по заголовку и тексту (без учёта регистра)

Сортировка по дате создания (новые сверху)

Сортировка по дате последнего изменения

3. Нефункциональные требования
Минимальная длина пароля — 8 символов

Валидация данных и на клиенте, и на сервере

4. Технологии
Фронтенд: HTML, CSS, обычный JavaScript (без фреймворков)

Бэкенд: Python + Flask (или FastAPI)

База данных: PostgreSQL

5. Основные страницы
Вход — форма с email и паролем

Регистрация — форма с именем, email, паролем и подтверждением

Главная — список заметок, кнопка создания, поле поиска, кнопка выхода

Просмотр заметки — полный заголовок и текст, кнопки редактирования и удаления

6. Требования к безопасности
Защита от SQL-инъекций (параметризованные запросы или ORM)

Защита от XSS (экранирование пользовательского ввода)

Хэширование паролей перед сохранением в БД

При попытке зайти на главную без авторизации — перенаправление на страницу входа

7. Критерии готовности
Пользователь может создать заметку и сразу увидеть её в списке

Можно отредактировать заметку — изменения отображаются

Можно удалить заметку — она исчезает из списка

Поиск находит нужные заметки по ключевым словам

После выхода из системы доступ к заметкам закрыт

Пароли в базе данных хранятся не в открытом виде

8. Базовые шаги разработки
Настройка окружения и установка библиотек

Создание базы данных и подключение к ней

Реализация регистрации и входа

Реализация операций с заметками (создание, чтение, редактирование, удаление)

Добавление поиска и сортировки

Вёрстка страниц (HTML/CSS)

Написание JavaScript для отправки запросов к серверу без перезагрузки страницы

Тестирование и отладка

9. Возможные улучшения (опционально)
Добавление тегов к заметкам

Экспорт заметок в файл

Пагинация (по 10-20 заметок на страницу)

Тёмная тема оформления# up_project
